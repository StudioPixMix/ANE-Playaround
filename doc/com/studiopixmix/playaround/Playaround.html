Packagecom.studiopixmix.playaround
Classpublic class Playaround
InheritancePlayaround Object

The Playaround extension. Before trying to do anything (even checking for compatibility), you must initialize the extension. Once done, use isCompatible() to check for compatibility with the current OS before trying to use Playaround features.

init()method 
public static function init(secretKey:String, useDefaultInstallPromptDialog:Boolean, debug:Boolean = false):void

Initializes Playaround extension with the given secret key. Call this only once in your app, before setting the current user or checking for extension compatibility.

Parameters

secretKey:String — Your Playaround secret key
 
useDefaultInstallPromptDialog:Boolean — Whether you will provide your own install prompt. If true, you will need to manually call didAcceptInstall() and didRefuseInstall()
 
debug:Boolean (default = false) — Whether debug mode should be enabled

SHOULD_DISPLAY_CUSTOM_INSTALL_PROMPTConstant
public static const SHOULD_DISPLAY_CUSTOM_INSTALL_PROMPT:String = Playaround.ShouldDisplayCustomInstallPrompt

Event dispatched by the extension when useDefaultInstallPromptDialog is set to false, and the Playaround SDK requires displaying a custom install prompt dialog. The dialog must complete either by calling didAcceptInstall() or didRefuseInstall().
